This repository contains a project developed during my studies (Master’s Degree in Physics at the University of Perugia).

The project investigates a potential correlation between 2019 pollution data and the transmission of Covid-19 in 2020
across the municipalities of Umbria, an Italian region. Naturally, no correlation was found.

This work was part of the exam for the "Data Science and Applications for Physics" course.
The theoretical part of the course focused on Big Data analysis methods, while the practical part covered Python programming
and the use of specific libraries for data analysis and machine learning.

File Descriptions:

    Esame.ipynb: A Jupyter Notebook containing the project (in Italian).

    dataset_exam.csv: The dataset containing the data to be processed.

    features_definition.docx: A document describing the data features, useful for understanding
                              the context and the meaning of the dataset's columns.

    limits_R_10_minicipalities.geojson: A file used by a specific Python library to plot the boundaries
                                        of the municipalities in Umbria.
